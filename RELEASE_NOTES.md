# Release Notes

## v1.0.0

Initial full-book release of `AnalyticCombinatorics`.

### Highlights

- 1,284 Lean source files.
- 264,166 lines of Lean source.
- Full library build passes with 4,548 jobs.
- Examples build passes with 3,305 jobs.
- No `sorry`, `admit`, or `axiom` remains in the Lean sources.
- Root import coverage includes every non-`Examples` module.
- GitHub Actions passes on the release branch.

### Coverage

- Part A: symbolic methods, labelled constructions, parameters, and examples.
- Part B: complex asymptotic methods, rational and meromorphic generating
  functions, singularity analysis, saddle-point methods, and limit laws.
- Appendices: finite models, complex analysis foundations, and
  Tauberian/probabilistic schemas.
- Examples: executable sanity checks for Fibonacci, Catalan families,
  derangements, set partitions, integer partitions, compositions, trees,
  surjections, strings, and related sequences.

Internal handoff/task logs are not included in the release tree.
